This page shows how to converts your excel questions into html form

1. How it works?

  1. Create a standard excel file as shown in example
  2. Click on choose file button to select your file
  3. After selecting your file click on create form button it'll create your form
  4. To view from click on view form button
  5. After viewing click on download button to download your form in txt file

2. The sample excel file is as show below

  1. It consists six columns namely class_Name, ID, Type_Of_Answer, Question, No_Of_Options and Answers.
    1. Where all column names are not case sensitive.
    2. The top row is fixed.
    3. Where the fields in all the columns are mandatory.
  2. The column Class_Name defines the class name of the question
  3. The column ID defines the id of the question
  4. The column Type_Of_Answer defines the type of answers for particular question
    • This column accept 3 fields (Multi_Choice_single_Answer, Multi_Choice_Multi_Answer and User_Input
    • Multi_Choice_Single_Answer gives radio type of options for question (Where only one option is true)
    • Multi_Choice_Multi_Answer gives checkbox type of options for a question (Where more than one options are true)
    • User_input gives a textfield where user need to enter answer
  5. The Question column defines the question
  6. No_Of_Options accept a number where it defines the number of options for particular column
  7. Options column defines the options for particular question
    Note- If the Type_Of_Answer is User_Input the field in options column should be answer
  8. The Options_Class column defines the class name for the options
  9. The column Options_ID defines the id of the options
